# Overview

This folder contains a mapping set that details the semantic and syntactic alignments between the Barcode Core Data Model metadata schema (used by BOLD) and the MIxS Core standard, following the Simple Standard for Sharing Ontological Mappings (SSSOM) framework (Matentzoglu et al., 2022). This work was developed within Task 3.2 of the Horizon Europe-funded eDNAqua-Plan project, which aims to improve interoperability by aligning community (meta)data standards across the full aquatic eDNA biomonitoring workflow, contributing to a sustainable and interoperable European eDNA digital ecosystem.

# Mapping Approach

To establish this mapping set, we followed the mapping approach described in the manuscript _Aligning Standards Communities for Omics Biodiversity Data: Sustainable Darwin Core-MIxS Interoperability_ (Meyer et al.,2023, https://bdj.pensoft.net/article/112420/), adapting it where necessary to maintain continuity with previously developed eDNA mappings. We adopted the semantic and syntactic mapping framework from this prior work, resulting in two tables, one for semantic mappings and one for syntactic mappings, which together form a single mapping set.

Because performing semantic and syntactic alignment simultaneously is more efficient in terms of time and coordination, this folder also contains a support mapping file that integrates both semantic and syntactic mappings to better support collaboration.

# Background and prior work

This mapping set also builds on the work developed within the Biodiversity Genomics Europe (BGE) project during the _Enhancing Digital Infrastructures and Data Handling Practices for Single Specimen Barcoding_ 2024 Barcoding Hackathon (https://doi.org/10.37044/osf.io/xb4ut_v1).The present mapping set updates the mapping effort established during that event and expands it to include all required minimal SSSOM slots, as well as both semantic and syntactic mappings (earlier work included semantic alignment only).
